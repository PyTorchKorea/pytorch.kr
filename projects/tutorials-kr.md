파이토치 한국 사용자 모임은 2018년부터 PyTorch 공식 문서를 한국어로 번역해 왔습니다.
한국어 사용자들이 언어 장벽 없이 PyTorch를 배우고 활용할 수 있도록,
튜토리얼, 모델 허브, 블로그 등 다양한 공식 자료를 번역하고 있습니다.

번역은 모두 오픈소스로 진행되며, 누구나 참여할 수 있습니다.
번역 시에는 [PyTorchKorea 번역 용어 가이드](https://github.com/PyTorchKorea/tutorials-kr/blob/master/TRANSLATION_GUIDE.md){:target="_blank"}를 따라
일관된 한국어 표현을 사용합니다.

## 튜토리얼 번역

<a href="https://tutorials.pytorch.kr" target="_blank" class="btn btn-primary btn-sm mr-1">
  <i class="fa fa-globe"></i> tutorials.pytorch.kr
</a>
<a href="https://github.com/PyTorchKorea/tutorials-kr" target="_blank" class="btn btn-outline-secondary btn-sm">
  <i class="fa fa-github"></i> GitHub
</a>

[PyTorch 공식 튜토리얼](https://docs.pytorch.org/tutorials/){:target="_blank"}을 한국어로 번역하는 프로젝트입니다.
PyTorch의 핵심 개념부터 고급 기법까지, 단계별로 학습할 수 있는 튜토리얼을 한국어로 제공합니다.
현재 **PyTorch v{{ site.data.tutorials_kr_version }}** 기준으로 번역이 진행되고 있습니다.

**주요 콘텐츠:**
- [PyTorch 기본 익히기](https://tutorials.pytorch.kr/beginner/basics/intro.html){:target="_blank"} — Tensor, Dataset, 학습 루프 등 핵심 개념
- 레시피(Recipes) — ML 워크플로우 구축을 위한 단계별 가이드
- 프로토타입 레시피 — 최신 기능들의 실험적 사용법

## 모델 허브 번역

<a href="https://pytorch.kr/hub" target="_blank" class="btn btn-primary btn-sm mr-1">
  <i class="fa fa-globe"></i> pytorch.kr/hub
</a>
<a href="https://github.com/PyTorchKorea/hub-kr" target="_blank" class="btn btn-outline-secondary btn-sm">
  <i class="fa fa-github"></i> GitHub
</a>

[PyTorch Hub](https://pytorch.org/hub/){:target="_blank"}에 등록된 사전 학습 모델(Pre-trained Model)들의 문서를
한국어로 번역하여 제공합니다. 다양한 딥러닝 모델의 사용법과 설명을 한국어로 쉽게 확인할 수 있습니다.

**주요 콘텐츠:**
- 이미지 분류 — ResNet, AlexNet, VGG, GoogLeNet 등
- 객체 탐지 — SSD, Faster R-CNN 등
- 자연어 처리 — Transformer 기반 모델
- 생성 모델 — GAN 등

## 블로그 번역

<a href="https://pytorch.kr/blog" target="_blank" class="btn btn-primary btn-sm mr-1">
  <i class="fa fa-globe"></i> pytorch.kr/blog
</a>
<a href="https://github.com/PyTorchKorea/pytorch.kr" target="_blank" class="btn btn-outline-secondary btn-sm">
  <i class="fa fa-github"></i> GitHub
</a>

[PyTorch 공식 블로그](https://pytorch.org/blog/){:target="_blank"}의 글을 한국어로 번역하여 최신 소식과 기술 동향을 전달합니다.
새로운 버전 출시, 기술 심층 분석, 생태계 소식 등 다양한 내용을 한국어로 읽을 수 있습니다.

**주요 콘텐츠:**
- 릴리즈 소식 — PyTorch 새 버전의 주요 변경사항과 새로운 기능
- 기술 심층 분석 — 커널 퓨전, 컴파일러 최적화 등
- 생태계 소식 — 새로운 도구와 라이브러리 소개

## 참여 방법

번역 프로젝트에 누구나 참여할 수 있습니다:

1. **번역하기** — 아직 번역되지 않은 문서를 한국어로 번역합니다.
2. **리뷰하기** — Pull Request된 번역이 적절한지 검토합니다.
3. **오류 제보** — 번역이 잘못되었거나 오래된 부분을 이슈로 제보합니다.

참여 시에는 각 프로젝트의 GitHub 저장소를 참고해주세요.
