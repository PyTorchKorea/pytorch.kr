PyTorch 한국어 튜토리얼 번역

GitHub tutorials.pytorch.kr

프로젝트 소개

PyTorch 한국어 튜토리얼 번역 프로젝트는 PyTorch 공식 튜토리얼을 한국어로 번역하여 제공하는 프로젝트입니다. 한국어를 사용하는 많은 분들이 PyTorch를 보다 쉽게 배우고 활용할 수 있도록, 파이토치 한국 사용자 모임에서 운영하고 있습니다.

번역된 튜토리얼은 tutorials.pytorch.kr에서 확인할 수 있으며, 현재 PyTorch v2.8 기준으로 번역이 진행되고 있습니다.

주요 콘텐츠

  • PyTorch 기본 익히기 — PyTorch의 핵심 개념과 모듈을 단계별로 학습합니다.
  • 레시피(Recipes) — 전체 ML 워크플로우를 구축하기 위한 단계별 가이드를 제공합니다.
  • TensorBoard 시각화 — 데이터 및 모델 학습 과정을 시각화하는 방법을 다룹니다.
  • 프로토타입 레시피 — 최신 기능들의 실험적 사용법을 소개합니다.

참여 방법

누구나 번역에 참여할 수 있습니다. 기여하는 방법은 크게 두 가지입니다:

  1. 번역하기 — 아직 번역되지 않은 튜토리얼을 한국어로 번역합니다.
  2. 리뷰하기 — Pull Request된 번역 문서가 적절한지 검토합니다.

자세한 기여 방법은 GitHub 저장소의 기여 가이드를 참고해주세요. 번역 시 사용하는 용어는 번역 용어 가이드를 따릅니다.
